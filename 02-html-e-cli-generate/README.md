# Aula 02 - Html e CLI `generate`

## Tópicos trabalhados:

- Conceitos HTML
- Trabalhando com CLI `generate`: `component` e `service`. 
- Como apresentar dados do TS no HTML

## Comandos utilizados

- `npm start`
- `ng generate component meu-primeiro-componente`
- `ng generate service exemplo`
